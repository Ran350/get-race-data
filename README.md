# 競馬データ収集

RCC 2020 年度競馬 AI 班　スクレイピング係

## 概要

以下の競馬データを収集する．

- 競走馬マスタ
- 騎手マスタ
- レースデータ
- 調教師マスタ

json 形式で出力する.

## 実行環境

- Python ~> 3.7
- requests 2.22.0
- beautifulsoup4 4.8.2

## 準備

```sh
git clone <this repo>
cd <this repo>
```

## 実行

- 競走馬マスタ -> horse/README.md
- 騎手マスタ -> jockey/README.md
- レースデータ -> race/README.md
- 調教師マスタ -> trainer/README.md

の指示に従って実行してください．
