# 騎手マスタ収集

## 実行

```sh
python main.py
```

## 出力形式

騎手マスタを取得

- 取得対象：現役，引退騎手
- 近走成績が 0 件の騎手の勝率と平均順位には 0 埋め
- 順位が数値でないものは除外
  (''(なし),'除','中','取','失','降','再')

## データ構造

```
[
    {
        "netkeiba_id":"５桁の騎手id(文字列型)",
        "name":"騎手名",
        "win_rate":"直近20件の勝率(小数)",
        "rank_average":"直近20件の平均順位(小数)"
    },
    {...},
    {...},
    ...
]
```
